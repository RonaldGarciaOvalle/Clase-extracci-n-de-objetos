Extracción Automatizada de algunos objetos cartográficos

Este repositorio contiene el flujo de trabajo para la detección y segmentación de construcciones y piscinas en pequeños recortes de ortoimágenes  satelitales utilizando modelos híbridos U-NET entrenados con datos locales.

El objetivo de este cuaderno es automatizar la segmentación de estos objetos con fines únicamente educativos y exploratorios, los cuales se ajustan a las pautas teóricas del curso de SIG AVANZADO en la temática de extracción de objetos.

Para utilizar esta ifnormación por favor abrir en Colab y ejecute según las pautas y el orden de la sesión.

Muchas gracias.
